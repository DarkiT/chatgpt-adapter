## 配置说明

```config.yaml
custom-llm:
  - prefix: github
    proxied: true
    reversal: https://models.inference.ai.azure.com
  - prefix: grok
    proxied: true
    reversal: https://api.x.ai/v1
```

`prefix` 前缀

`proxied` 是否使用代理

`reversal` 转发地址

例子： `github/gpt_4o` `grok/grok-beta`

## 模型列表

```json
[
    "${prefix}/xxx"
]
```

## 请求示例
